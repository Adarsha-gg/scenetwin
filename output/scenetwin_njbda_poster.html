Reference-free evaluation of audio description quality for blind and low-vision video access

Frame-grounded comprehension QA plus video-text grounding ranks audio descriptions without requiring a human-written reference answer.

Problem

Blind and low-vision viewers rely on audio descriptions to experience the same feeling as actually watching a video, but high-quality review is slow, subjective, and hard to scale.

Common automated scores need a reference caption or reward surface-level similarity. AD quality is different: a good description must help a listener answer the visual questions that matter.

Goal: score candidate AD text from the video itself, with no human reference description.

Method

SceneTwin combines two complementary signals, normalized within each clip and averaged 50/50.

CLIP-L14 grounding Cosine similarity between candidate AD text and sampled video frames. Catches wrong or hallucinated descriptions.
Frame-grounded ADQA LLM writes visual comprehension questions from frames. A second LLM blindly grades candidate descriptions.
Video framesSample the visual evidence directly from the clip.
→
Questions + groundingGenerate ADQA questions and CLIP frame-text scores.
→
Tier rankingScore each AD candidate against expected human quality order.

Why It Works

CLIP asks whether the description is visually grounded. ADQA asks whether the description supports comprehension.

Failure modeDetected by
Wrong scene or hallucinated actionCLIP-L14 grounding
Vague but visually plausible textFrame-grounded ADQA
Overly generic caption that misses key factsADQA questions
Model-specific grader noiseCross-model / multi-judge checks

Takeaway

Reference-free AD evaluation is feasible when visual grounding and listener comprehension are scored together.
